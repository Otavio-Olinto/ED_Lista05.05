# ED_Lista05.05
## Aplicando o algortimoa QuickSort

Este desafio consiste em criar uma aplicação, em Java, que utilize uma biblioteca de QuickSort para ordenar os seguintes vetores:

```
Vetor1: 74 20 74 87 81 16 25 99 44 58
Vetor2: 44 43 42 41 40 39 38
Vetor3: 31 32 33 34 99 98 97 96
```

```
Requisitos do projeto:

1. Arquitetura MVC;
2. Os vetores devem ser definidos na Main;
3. As demais operações devem ser realizadas por classes no pacote controller.
